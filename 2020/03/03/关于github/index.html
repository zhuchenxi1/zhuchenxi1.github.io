    一些名词

  • Followers:粉丝

  • watch : 对于别人的仓库默认是not watching。改成watching之后就可以收到这个仓库的动态

  • Star :类似于喜欢

  • Fork : 克隆仓库，用于之后pr

  • Issues:类似留言区。 new issue提出问题，可以由自己或项目所有者关闭。关闭的问题仍可被看到。

  • pull requests : 对开源项目做贡献

  • Projects : 项目进度管理

  • Wiki : 对项目的介绍

  • Insights : 项目的统计信息

  • Settings

    • GitHub pages : 提供网站服务
    • Webhooks :
    • danger zone : 可以将仓库变成private / public 或者删除仓库
